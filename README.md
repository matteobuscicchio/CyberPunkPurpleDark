# CyberPunkPurpleDark

A cyberpunk-inspired, purple-based dark theme for Visual Studio Code.

## Features

- Vibrant cyberpunk color palette with purples and neon accents
- Custom backgrounds and borders for editor, sidebar, panels, and tabs
- Enhanced status bar, activity bar, and title bar styling
- Carefully tuned token colors for code readability
- Based on the default Dark+ theme, with extensive customizations

## Credits

This theme is inspired by and based on the default [Dark+](https://github.com/microsoft/vscode) theme for Visual Studio Code, with customizations by [Matteo Buscicchio](https://github.com/matteobuscicchio).
The color palette is directly inspired by the official Cyberpunk website, but this theme is an independent project and is not affiliated with or endorsed by CD Projekt RED.
